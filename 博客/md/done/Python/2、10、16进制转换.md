# 16 ↔ 10

## 16 → 10

第一个参数是字符串 '0Xff' ,第二个参数是说明，这个字符串是几进制的数。


```python
int('0xf',16)  #0x加不加皆可
```




    15



## 16 ← 10


```python
hex(15)
```




    '0xf'



# 2 ↔ 10

## 2 → 10


```python
int('1111',2)   
```




    15



## 2 ← 10


```python
bin(15)
```




    '0b1111'



# 2 ↔ 16
先叨下二进制以及十六进制中的快速口算方法
(1101111) = (110 1111) = (0110 1111)
(0110) = 0x6
(1111) = 0xf
(1101111) = 0x6f

## 2 → 16


```python
hex(int('1101111',2))
```




    '0x6f'



## 2 ← 16


```python
 bin(int('6f',16))
```




    '0b1101111'
